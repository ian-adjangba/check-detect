# Check Detect Proposal

## Project Title
Check Detect: An AI-Assisted Check Fraud Review Tool for Teller Operations

## Summary
Check Detect is a proposed AI-assisted fraud detection support tool for use in teller operations. The system is designed to help bank tellers evaluate suspicious checks by analyzing a combination of check-material characteristics, numbering patterns, payer geography, and other structural or contextual risk indicators. Rather than making automated decisions, Check Detect generates a legitimacy likelihood score and explains the reasons behind its assessment, allowing tellers to use the output as a second set of eyes during check processing.

## Problem
Banks face losses from counterfeit checks, check kiting, and fraudulent instruments that may initially clear but later return unpaid or invalid. Tellers often have limited time and incomplete information when making frontline decisions.

## Objective
The goal of Check Detect is to provide explainable AI-assisted risk scoring for suspicious checks while preserving human discretion.

## Proposed Method
- Use synthetic metadata to simulate check review scenarios
- Engineer fraud-related features
- Combine rule-based logic with a baseline machine learning model
- Present the output as a risk score and explanation list for teller review

## Deliverables
- Synthetic fraud dataset
- Baseline scoring engine
- Prototype interface
- Project documentation and ethics notes
