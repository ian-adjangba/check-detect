# Check Detect — AI Fraud Detection System

Live App: https://check-detect-scvukwwu73opgrfbnvvpjh.streamlit.app/  
Kaggle Workflow: https://www.kaggle.com/code/ianadjangba/check-detect-workflow  

Check Detect is an AI-assisted decision-support tool designed to help bank tellers identify suspicious checks using rule-based scoring and machine learning.

## Key Features
- Real-time fraud risk scoring
- Streamlit dashboard for teller use
- Synthetic dataset simulating real banking fraud
- Explainable AI outputs (audit-friendly)

## Use Case
Designed to reduce:
- Check kiting
- Counterfeit check fraud
- Returned fraudulent deposits

## Tech Stack
Python, Pandas, Scikit-Learn, Streamlit
