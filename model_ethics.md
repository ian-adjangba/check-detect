# Model Ethics and Usage Notes

Check Detect is intended only as a decision-support system. It must not automatically approve or deny a customer transaction.

## Key Principles
- Human tellers remain responsible for the final decision
- Risk indicators are advisory, not conclusive proof of fraud
- Geographic signals should only be used as contextual inputs
- The model should be evaluated for false positives and fairness concerns
- Production use would require compliance review, validation, and data governance controls
